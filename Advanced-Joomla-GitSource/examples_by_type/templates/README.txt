3- What is included in the source code?

A variety of extensions including:
	* mod_twitter -- Module that uses the Twitter API to search using Twitter keywords and display relevant tweets
	* mod_bingnews -- Modules that searches the Bing news API and displays relevant new items
	* mod_flickr -- Searches the Flickr API for images that match a search term
